# EDmail

**Français** | [English](#english)

---

## Français

EDmail est une application de messagerie électronique disponible sur Windows, Linux, macOS et Android.

### Fonctionnalités

- Consultation et envoi d'e-mails
- Gestion de plusieurs comptes
- Interface réactive adaptée bureau et mobile
- Notifications de nouveaux messages
- Mise à jour automatique intégrée

### Plateformes

| Plateforme | Format |
|------------|--------|
| Windows | `.exe` (installeur NSIS) |
| Linux | `.tar.gz` |
| macOS | `.dmg` |
| Android | `.apk` |

### Installation

Télécharge la dernière version depuis les [releases GitHub](https://github.com/3yezz/mail/releases).

### Stack technique

- **Interface** : React 18
- **Bureau** : Electron
- **Mobile** : Android natif (WebView)
- **Build** : electron-builder, Gradle

---

## English

EDmail is an email client application available on Windows, Linux, macOS and Android.

### Features

- Read and send emails
- Multiple account management
- Responsive interface for desktop and mobile
- New message notifications
- Built-in auto-update

### Platforms

| Platform | Format |
|----------|--------|
| Windows | `.exe` (NSIS installer) |
| Linux | `.tar.gz` |
| macOS | `.dmg` |
| Android | `.apk` |

### Installation

Download the latest version from the [GitHub releases](https://github.com/3yezz/mail/releases).

### Tech stack

- **UI** : React 18
- **Desktop** : Electron
- **Mobile** : Android native (WebView)
- **Build** : electron-builder, Gradle
